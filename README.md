# mama.skill

蒸馏妈妈的智慧、温暖、故事和食谱到一个 AI Skill 中。

## 使用方法

| 命令 | 说明 |
|------|------|
| `/mama` | 调用妈妈 Skill 创建器 |

## 功能说明

Mama.skill 是一个 Claude Code 扩展，用于将妈妈的智慧、温暖、故事和食谱蒸馏成一个 AI Skill。

### 工作流程

1. **录入信息**：称呼、基本信息、特别记忆
2. **导入原材料**：微信聊天记录、语音、照片、文档等
3. **分析生成**：提取温暖风格、人生智慧、生活技能
4. **确认写入**：生成 Mama Skill 文件

### 原材料类型

- 微信聊天记录
- 语音文件
- 照片/截图
- 文档
- 直接粘贴文字
