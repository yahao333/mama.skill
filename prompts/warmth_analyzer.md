# 温暖风格提取器

## 任务

从用户提供的原材料中，提取妈妈的说话风格、表达方式、关心人的独特方式。

## 提取维度

### 1. 称呼习惯

她平时怎么称呼家人？
- 对晚辈的称呼（宝宝/宝贝/小名...）
- 对平辈的称呼
- 特殊昵称

```
## 称呼习惯

称呼用户：{她怎么叫你}
称呼配偶：{怎么称呼老伴}
称呼其他人：{爷爷奶奶/亲戚...}
```

### 2. 语气特征

从材料中分析她的语气特点：
- 常用语气词（哎呀/哎呦/宝贝儿...）
- 句式特点（倒装句/感叹句...）
- 关心人的方式（直接关心/委婉关心/行动关心...）

```
## 语气特征

语气词：{她常用的语气词}
关心方式：{她怎么表达关心}
责备方式：{她怎么温和地批评}
鼓励方式：{她怎么给人打气}
```

### 3. 情感表达

分析她在不同场景下的情感表达方式：
- 开心时：{怎么表达喜悦}
- 担心时：{怎么表达关心}
- 生气时：{怎么表达不满}
- 思念时：{怎么表达想念}

```
## 情感表达

喜悦表达：{}
担心表达：{}
责备表达：{}
思念表达：{}
```

### 4. 独特关心方式

从材料中找出她独特的关心人的方式：
- 特定的食物/饮品关怀
- 特定的话语关怀
- 行动关怀

```
## 独特关心方式

身体不适时：{她怎么照顾}
心情不好时：{她怎么安慰}
取得成绩时：{她怎么鼓励}
遇到困难时：{她怎么支持}
```

---

## 输出要求

- 保持她的原话和语气风格
- 优先提取有具体场景/对话支撑的内容
- 特别注意方言、口头禅的表达
- 输出要让用户读起来有「这就是她」的感觉
