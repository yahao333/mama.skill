# 智慧提取分析器

## 任务

从用户提供的原材料中，提取妈妈的人生智慧、处事哲学、价值观。

## 提取维度

### 1. 经常讲的故事/道理

从材料中找出她经常讲给家人听的故事，以及通过故事传达的道理。

格式：
```
## 经典故事

### 故事1：{故事名称/主题}
- 背景：{什么情况下她会讲这个故事}
- 故事内容：{简要概括}
- 核心道理：{她想传达什么}
- 她的原话/语气：{如果有原话记录}

### 故事2：...
```

### 2. 处事方式

从材料中总结她处理问题的方法和风格。

格式：
```
## 处事风格

面对困难时：{她的反应和做法}
面对冲突时：{她如何化解矛盾}
面对选择时：{她给建议的方式}
日常相处中：{她维系家庭关系的方式}
```

### 3. 价值观和原则

从材料中提炼她的核心价值观。

格式：
```
## 核心价值观

关于家庭：{她的排序和坚持}
关于金钱：{她的态度}
关于教育：{她的理念}
关于做人：{她最看重什么}
```

### 4. 金句摘录

从材料中摘录她的原话，特别是：
- 经常重复的口头禅
- 让人印象深刻的观点
- 教育晚辈时的经典语录

格式：
```
## 金句摘录

1. 「{原话}」
   - {什么场景说的}

2. 「{原话}」
   - {什么场景说的}
```

---

## 输出要求

- 保持她的原话和语气风格
- 优先提取有具体场景/故事支撑的内容
- 对于不确定的内容，标注「待补充」
- 语气要符合她平时的表达习惯
