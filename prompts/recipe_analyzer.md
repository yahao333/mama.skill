# 食谱提取生成器

## 任务

从用户提供的原材料中，提取妈妈的厨艺精华：拿手菜、烹饪技巧、家传秘方。

## 提取维度

### 1. 拿手菜系列

从材料中找出她最拿手的菜品：

```
## 拿手菜

### 菜品1：{菜名}
- 特点：{这道菜的独特之处}
- 原料：{主要食材}
- 步骤要点：{关键步骤}
- 小技巧：{她的独门心得}
- 背后的故事：{这道菜的意义}

### 菜品2：...
```

### 2. 烹饪风格

分析她的整体烹饪风格：

```
## 烹饪风格

口味偏好：{清淡/重口/甜/辣...}
烹饪方式：{炒/炖/蒸/煮...}
配料特点：{常用什么配料}
摆盘风格：{讲究/随意...}
```

### 3. 独门秘方/家传配方

特别记录那些没有写进食谱、只有她知道的秘诀：

```
## 独门秘方

### 秘方1：{名称}
- 秘诀内容：{具体的配方或技巧}
- 注意事项：{她特别叮嘱的点}
- 传授场景：{她是怎么教你的}
```

### 4. 节日/特殊场合美食

她在不同节日或场合会做什么特别的食物：

```
## 节日美食

春节：{}
端午节：{}
中秋节：{}
生日：{}
其他特殊场合：{}
```

### 5. 食材保存与处理

她的食材处理和保存智慧：

```
## 食材智慧

买菜技巧：{怎么挑选食材}
保鲜方法：{她怎么保存}
处理技巧：{食材预处理}
```

---

## 输出要求

- 步骤要具体到能实操的程度
- 保留她的原话描述
- 特别注意那些「适量」「少许」等经验性描述
- 可以补充一些她在厨房里的习惯性动作
