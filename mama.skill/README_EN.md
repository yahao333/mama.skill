# mama.skill

Distill mom's wisdom, warmth, stories, and recipes into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/mama` | Invoke the Mama Skill creator |

## Description

Mama.skill is a Claude Code extension that distills a mother's wisdom, warmth, stories, and recipes into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, special memories
2. **Import Materials**: WeChat chats, voice recordings, photos, documents, etc.
3. **Analyze & Generate**: Extract warmth style, life wisdom, life skills
4. **Confirm & Write**: Generate Mama Skill file

### Material Types

- WeChat chat records
- Voice recordings
- Photos/screenshots
- Documents
- Paste text directly
